Stage 0 notes

Webpages: Text doc written using HTML – hypertext markup language

Coding: Telling computers what to do in a language they understand – e.g. through HTML.

Computers are stupid: Coding in language with strict rules implies the computer that interprets is prone to misunderstand when the human coder makes errors. There’s no “human factor” that understands the mistake and the computer act stupid when interpreting an error.

Look it up: Too much detail. No problem, focus on the larger ideas and look up the details (correct html tags etc.) on the internet.0

HTML lingo: Tags (some require closing brackets, some don’t), inline tags, block tags, void tags, elements, attributes

HTML structure: Doctype, head (meta-data, css, javascript), body (the fun stuff)

Img tags are inline.
